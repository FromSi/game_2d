## Установка

* Установить Go 1.21.5 или выше
* Установить зависимости https://ebitengine.org/en/documents/install.html
* Проверка окружения на правильную работу Ebitengine 
* `go mod tidy`

Для запуска `go run cmd/app/main.go`
